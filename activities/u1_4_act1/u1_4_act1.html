Unit 1.4 · Activity 1 · Rally CoachSolution 3

Solution 3

Verify equivalence
Use substitution with $x=4$ to check whether $2(x+6)$ and $2x+12$ are equivalent.
Solution: For $x=4$, $2(x+6)=2(4+6)=2(10)=20$. Also, $2x+12=2(4)+12=8+12=20$. They give the same value for $x=4$.

Unit 1.4 · Activity 1 · Rally CoachSolution 4

Solution 4

Structure explanation
Explain why $4(x+3)+2(x+3)$ can be rewritten as $6(x+3)$ before expanding.
Solution: Both terms contain the same factor, $(x+3)$. There are $4$ groups of $(x+3)$ plus $2$ more groups of $(x+3)$, so there are $6$ groups total: $6(x+3)$.

Solution 10

Context equation
A phone plan costs $15$ dollars per month plus a $10$ activation fee. Write and solve an equation to find how many months cost $70$ dollars total.
Solution: Let $m$ be the number of months. The equation is $15m+10=70$. Subtract $10$: $15m=60$. Divide by $15$: $m=4$ months.

Problem 11

Strategy comparison
Compare solving $3(x+2)=21$ by first dividing by $3$ with solving by first distributing. Which is more efficient here?
Unit 1.4 · Activity 1 · Rally CoachSolution 11

Solution 11

Strategy comparison
Compare solving $3(x+2)=21$ by first dividing by $3$ with solving by first distributing. Which is more efficient here?
Solution: Dividing first is more efficient: $3(x+2)=21$ gives $x+2=7$, so $x=5$. Distributing also works: $3x+6=21$, so $3x=15$ and $x=5$, but it takes more steps.

Unit 1.4 · Activity 1 · Rally CoachSolution 12

Solution 12

Create two strategies
Create an equation that can be solved efficiently in two different ways. Show both strategies and explain why they give the same solution.
Solution: Answers vary. Example: $2(x+4)=18$. Divide first: $x+4=9$, so $x=5$. Distribute first: $2x+8=18$, so $2x=10$ and $x=5$. They give the same solution because distributing creates an equivalent equation.
