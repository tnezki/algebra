Station 1: Function Families Solutions

Question 1

Prompt: Use the graph set. Which graph appears to be exponential decay?

Solution: Graph C appears to be exponential decay. It decreases quickly at first and then levels off as $x$ increases.

Question 2

Prompt: Classify $y=3(2)^x$ by function family and describe what happens to the outputs as $x$ increases by $1$.

Solution: It is exponential growth. Each time $x$ increases by $1$, the output is multiplied by $2$.

Question 3

Prompt: Identify the family shown in the graph and justify your answer with at least two visible features.

Solution: The graph shows an absolute value function. It has a V-shape, a vertex at about $(-1,-2)$, and two straight-line pieces.

Question 4

Prompt: Create a four-card classification challenge: one equation, one table, one graph description, and one context. Make all four cards represent different function families. Write a clue for each that does not name the family.

Solution: Sample: equation $y=2x+1$ with clue “outputs add $2$ each step”; table $x:0,1,2,3$ and $y:1,4,9,16$ with clue “first differences change by a constant amount”; graph description “V-shape with vertex $(0,-2)$”; context “a population doubles every hour.”

Station 3: Expression Structure Solutions

Question 9

Prompt: Rewrite $8x+20$ by factoring out the greatest common factor.

Solution: The greatest common factor is $4$, so $8x+20=4(2x+5)$.

Question 10

Prompt: Use substitution with $x=3$ to check whether $4(x-2)+5$ and $4x-3$ are equivalent.

Solution: When $x=3$, $4(x-2)+5=4(1)+5=9$. Also, $4x-3=12-3=9$. They match for $x=3$.

Question 11

Prompt: Use structure to explain why $5x+3x$ can be simplified but $5x+3$ cannot.

Solution: $5x$ and $3x$ are like terms because both are $x$-terms, so they combine to $8x$. In $5x+3$, the $3$ is a constant, so it is not like $5x$.

Question 12

Prompt: Write incorrect student work for simplifying $3(2x+5)-4x$, then critique and revise it so the final expression is correct.

Solution: Example incorrect work: $3(2x+5)-4x=6x+5-4x=2x+5$. The error is that $3$ was not distributed to $5$. Correct work: $3(2x+5)-4x=6x+15-4x=2x+15$.

Station 4: Rate of Change Solutions

Question 13

Prompt: Determine the constant change in the outputs $11,16,21,26$.

Solution: The constant change is $+5$.

Question 14

Prompt: Compare the rates of change for Pattern A and Pattern B.

Solution: Pattern A has rate of change $+3$. Pattern B has rate of change $+5$. Pattern B grows faster by $2$ tiles per figure.

Question 15

Prompt: Use Pattern B. Explain why this graph does not show constant rate of change.

Solution: The graph does not show constant rate of change because the output changes are not the same. The outputs $1,2,4,8,16$ have changes $+1,+2,+4,+8$. The graph is curved, not a straight line.

Question 16

Prompt: Create two tile patterns that both have $12$ tiles at Figure 3 but have different constant rates of change. Make a small table for each and explain the difference.

Solution: Sample: Pattern A could have $x:0,1,2,3$ and $y:3,6,9,12$, so its rate is $+3$. Pattern B could have $x:0,1,2,3$ and $y:6,8,10,12$, so its rate is $+2$. Both have $12$ tiles at Figure 3, but Pattern A grows faster.
