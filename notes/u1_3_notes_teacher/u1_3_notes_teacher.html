Function Families
Teacher Notes
Recognizing the kind of relationship before doing every calculation.

Mathematical idea: A function family is like a pattern family. The members may look different, but they share important behavior.

Today we will sort common function families using graphs, tables, equations, and contexts. The goal is classification and reasoning, not full modeling yet.

Learning Target

Intro

Directions

Read the introduction aloud with a partner or in a small group. As you read, annotate the text by highlighting important ideas, circling key vocabulary, and writing short notes about connections, questions, or predictions in the margins.

A function family is a group of functions that share a common shape or pattern. Families help us recognize what kind of relationship we are looking at before we know every detail.

Some function families are easy to spot from a graph. A linear function makes a straight line. A quadratic function makes a U-shape called a parabola. An absolute value function often makes a V-shape. An exponential function curves because it changes by multiplying instead of adding the same amount.

Tables and equations can also give clues. A table with the same change each step often points to a linear relationship. An equation with $x^2$ often points to a quadratic relationship. An equation with the variable in the exponent, such as $2^x$, points to an exponential relationship.

Today we are not trying to prove a perfect model for every situation. We are learning to classify, compare, and explain evidence. A good classification answer says more than the family name. It explains what feature helped you decide.

Stop and Discuss

  1. Why might it help to recognize a function family before doing calculations?
  2. What is one graph clue and one equation clue from the reading?
  3. Why should a classification answer include evidence, not just a family name?
🔴 Sample student responses:
  • Function families are groups with similar shapes or patterns.
  • Linear: straight; quadratic: U-shaped; absolute value: V-shaped; exponential: curved growth/decay.
  • Graphs, equations, tables, and contexts all give clues.
🔴 Teacher Note: Listen for “evidence” language rather than guessing from appearance only.

Graphs Give Visual Clues

Many function families have a shape that is useful for a first classification. The shape is not the whole story, but it is a strong place to start.

Linear: straight-line behavior

Quadratic: U-shaped parabola behavior

Absolute value: V-shaped behavior

Exponential: curved growth or decay behavior

Example 1

Match each graph to the most likely function family: linear, quadratic, absolute value, or exponential. Then write one piece of evidence for each match.

🔴 Complete worked solution:
  • Straight-line graph: linear.
  • U-shaped graph: quadratic.
  • V-shaped graph: absolute value.
  • Curved growth/decay graph: exponential.
🔴 Teacher Note: What students usually do wrong: call any increasing graph linear. Emphasize shape and type of change.

You Try It 1

A graph is shaped like a V and has one clear turning point. Which function family is most likely? Explain the graph evidence you used.

🔴 Answer:

Most likely absolute value because the graph is V-shaped and has one clear turning point.

🔴 Teacher Note: Quick CFU: “What shape would make you think quadratic instead?” Extension: compare V-shape and U-shape.

Stop and Discuss

  1. Which graph family feels easiest to recognize? Why?
  2. Which two families could be mixed up if you only looked quickly?
🔴 Sample student responses:
  • Linear may feel easiest because it is a straight line.
  • Quadratic and exponential can be mixed up if students only notice both are curved.
🔴 Teacher Note: Connect to prior graph reading: shape is useful, but students must cite evidence.

Equations Give Structure Clues

An equation can show the family by how the input is used. Look for structure: Is $x$ being multiplied by a constant, squared, inside an absolute value, or used as an exponent?

Structure clue: A part of an equation that helps you recognize how the function behaves.

Example 2

Classify each equation by function family. Then underline or describe the structure clue.

  1. $y=5x-2$
  2. $y=x^2+4$
  3. $y=|x-3|+1$
  4. $y=3\cdot 2^x$
🔴 Complete worked solution:
  • $y=5x-2$: linear; $x$ is multiplied by a constant and not squared or an exponent.
  • $y=x^2+4$: quadratic; $x$ is squared.
  • $y=|x-3|+1$: absolute value; $x$ is inside absolute value bars.
  • $y=3\cdot2^x$: exponential; $x$ is in the exponent.
🔴 Teacher Note: Key point: structure tells the family. Do not let students classify by the last number only.

You Try It 2

A student says $y=4^x+1$ is linear because it has a $+1$ at the end. Critique the claim using the structure of the equation.

🔴 Answer:

The claim is incorrect. $y=4^x+1$ is exponential because the input $x$ is used as an exponent. The $+1$ shifts the outputs but does not make the relationship linear.

🔴 Teacher Note: Common misconception: any equation with “$+1$” or a simple number at the end is linear.

Tables Show How Outputs Change

A table can show whether the outputs are changing by adding a steady amount or by multiplying by a steady factor. This does not tell every detail, but it can give strong evidence for a family.

Example 3

Use the tables to decide which pattern looks more linear and which pattern looks more exponential. Explain using how the outputs change.

Table A

$x$$0$$1$$2$$3$$4$
$y$$2$$5$$8$$11$$14$

Table B

$x$$0$$1$$2$$3$$4$
$y$$1$$2$$4$$8$$16$
🔴 Complete worked solution:
  • Table A is linear because outputs add $3$ each time.
  • Table B is exponential because outputs multiply by $2$ each time.
🔴 Teacher Note: Emphasize “same add” versus “same multiply.” Students may only look at whether outputs increase.

You Try It 3

A table has outputs $10$, $15$, $20$, $25$, and $30$ as the inputs increase by $1$. Which family is most likely? Explain what stays the same.

🔴 Answer:

Most likely linear. The outputs increase by a constant amount: $+5$ each time.

🔴 Teacher Note: Quick CFU: “What would the outputs need to do for exponential evidence?”

Stop and Discuss

  1. How is adding the same amount different from multiplying by the same factor?
  2. Why does a table give evidence instead of a full proof from just a few points?
🔴 Sample student responses:
  • Adding the same amount gives constant differences.
  • Multiplying by the same factor gives constant ratios.
  • A few table values give evidence, but not always a complete proof of the whole function.
🔴 Teacher Note: Connect this to modeling: patterns need assumptions if they continue beyond listed data.

Classification Is Not Full Modeling

When we classify a function family, we decide what type of behavior seems most likely. A full model would need more information, like an exact equation, exact starting value, or a reason the pattern should continue.

Example 4

A student sees the graph below and says, “This is definitely linear because it goes up as $x$ increases.” Critique the claim using graph evidence.

🔴 Complete worked solution:

The claim is weak. A graph can increase without being linear. Linear graphs must increase or decrease at a constant rate and form a straight line. This graph curves, so it is not definitely linear.

🔴 Teacher Note: Key point: “goes up” describes direction, not family.

You Try It 4

A student says, “If I can name the family, then I know the exact equation.” Do you agree? Explain the difference between classifying a family and finding a full model.

🔴 Answer:

Disagree. Naming the family tells the general type of behavior, but it does not give the exact equation. Many functions can belong to the same family and have different starting values, rates, shifts, or scales.

🔴 Teacher Note: Extension: ask students to name two different linear equations.

Stop and Discuss

  1. What evidence could show that a graph is increasing but not linear?
  2. What extra information would you need to write an exact equation?
🔴 Sample student responses:
  • A graph can be increasing but not linear if it curves or changes by different amounts.
  • To write an exact equation, we need details such as points, starting value, rate, vertex, or multiplier.
🔴 Teacher Note: Listen for students separating classification from full modeling.

Summary

  • A function family is a group of functions with shared behavior.
  • Linear functions have straight-line behavior.
  • Quadratic functions often have U-shaped parabola behavior.
  • Absolute value functions often have V-shaped behavior.
  • Exponential functions often curve because values multiply by a factor.
  • Graphs, tables, equations, and contexts can all give classification evidence.
  • Classification helps you identify the type of relationship, but it is not the same as finding a complete model.

Common Mistakes

  • Using only one clue: A graph shape is helpful, but tables, equations, and contexts can support or challenge your choice.
  • Calling every increasing graph linear: Linear graphs are straight. Curved graphs can also increase.
  • Mixing up quadratic and absolute value: Quadratic graphs are smooth U-shapes. Absolute value graphs are sharper V-shapes.
  • Thinking repeated addition and repeated multiplication are the same: Adding the same amount often points to linear behavior. Multiplying by the same factor often points to exponential behavior.
  • Confusing classification with a full model: Naming the family does not automatically give the exact equation.
