Intro

Directions

Read the introduction aloud with a partner or in a small group. As you read, annotate the text by highlighting important ideas, circling key vocabulary, and writing short notes about connections, questions, or predictions in the margins.

Quadratic models often describe height, area, profit, or other quantities that rise and fall. The vertex can represent a maximum or minimum. Intercepts can represent starting values, break-even points, or when an object reaches the ground.

The same quadratic graph may include values that do not make sense in the context. Negative time, negative length, or impossible prices can appear algebraically. A restricted domain keeps only the input values that fit the situation.

Different forms reveal different features. Factored form shows zeros, vertex form shows the maximum or minimum, and standard form can be useful for formula work. Choosing the form depends on what the question asks.

Reasonableness checks connect the math back to the situation. A solution should be checked against units, graph features, and any limits in the context. This helps separate mathematically possible values from meaningful answers.

This lesson connects models, graphs, tables, vertices, intercepts, and contextual conclusions. Strong modeling explains what each feature means and why the answer is reasonable.

Quadratic models often use different graph features to answer different questions.

The question determines which feature is most useful.

Example 1: Choose the Useful Feature

Identify the feature most useful for each context.

  1. For $h(t)=-16t^2+64t$, which feature helps find maximum height?
  2. For $A(x)=x(18-x)$, which feature helps maximize area?
  3. For a break-even model, which feature identifies break-even inputs?

Teacher answer: a) Vertex. b) Vertex. c) x-intercepts or zeros. Reminder: ask students to justify each algebra move, not just state the answer.

Math Notes Projectile Height Model

A height model often has time as the input and height as the output.

$$h(t)=-16t^2+48t+4$$

The positive zero estimates when the object hits the ground. The negative zero is usually not meaningful for time after launch.

Example 2: Interpret a Landing Time

A ball has height $h(t)=-16t^2+48t+4$. Find when it hits the ground.

  1. Set the height equal to zero.
  2. Use a solving method that gives a positive time.
  3. Explain why one solution may not be meaningful.

Teacher answer: Solve -16t^2+48t+4=0. Positive solution is approximately t=3.08 seconds; the negative solution is not meaningful after launch. Reminder: ask students to justify each algebra move, not just state the answer.

You Try It 2: Interpret a Landing Time

A ball has height $h(t)=-16t^2+32t+12$. Find when it hits the ground.

  1. Set the height equal to zero.
  2. Use a solving method that gives a positive time.
  3. Explain why one solution may not be meaningful.

Teacher answer: Solve -16t^2+32t+12=0; positive solution is t=\frac{2+\sqrt{7}}{2}\approx2.32 seconds. The negative solution is not meaningful after launch. Follow-up: have students compare the structure to the example before checking work.

Stop and Discuss

  1. Why is the negative time solution usually rejected?

Teacher discussion: Negative time is before the start of the modeled event. Misconception watch: students may get a correct value without explaining what it means.

Math Notes Optimization with Area

Optimization asks for the maximum or minimum value of a quadratic model.

$$A(x)=x(30-2x)=-2x^2+30x$$ $$x=\frac{-b}{2a}=\frac{-30}{2(-2)}=7.5$$

$7.5$$15$$112.5$

Context: The vertex gives the dimensions that maximize area.

Example 3: Maximize Area

For $A(x)=x(30-2x)$, find the dimensions that maximize area.

  1. Rewrite the model in standard form.
  2. Find the input at the vertex.
  3. State the dimensions and maximum area.

Teacher answer: A(x)=-2x^2+30x; vertex input x=7.5. Other dimension is 15; maximum area is 112.5 square units. Reminder: ask students to justify each algebra move, not just state the answer.

You Try It 3: Maximize Area

For $A(x)=x(28-2x)$, find the dimensions that maximize area.

  1. Rewrite the model in standard form.
  2. Find the input at the vertex.
  3. State the dimensions and maximum area.

Teacher answer: A(x)=-2x^2+28x; vertex input x=7. Other dimension is 14; maximum area is 98 square units. Follow-up: have students compare the structure to the example before checking work.

Stop and Discuss

  1. How did the vertex answer the maximum-area question?

Teacher discussion: The vertex gives the maximum output of a downward-opening area quadratic. Misconception watch: students may get a correct value without explaining what it means.

Math Notes Reasonable Domain and Solutions

A model conclusion should name which solutions are meaningful and why.

$$h(t)=-16(t+0.2)(t-3.4)$$
SolutionContext decision
$t=-0.2$not meaningful after launch
$t=3.4$landing time

The domain might be $0\le t\le 3.4$ because the model describes height from launch until landing.

Example 4: Decide Which Solution Is Meaningful

A height model gives solutions $t=-0.1$ and $t=4.2$ seconds.

  1. Decide which solution is meaningful.
  2. State a reasonable time domain for the model.
  3. Explain your reasoning using the context.

Teacher answer: t=4.2 seconds is meaningful as the landing time. A reasonable domain is 0\le t\le4.2. Negative time is before launch. Reminder: ask students to justify each algebra move, not just state the answer.

You Try It 4: Decide Which Solution Is Meaningful

A profit model gives break-even prices $p=-3$ and $p=11$.

  1. Decide which solution is meaningful.
  2. State a reasonable price domain.
  3. Explain your reasoning using the context.

Teacher answer: p=11 is meaningful; p=-3 is not a real price. A reasonable domain could be 0\le p\le11 or a context-specific nonnegative interval. Follow-up: have students compare the structure to the example before checking work.

Stop and Discuss

  1. Why do domains and units matter when interpreting model solutions?

Teacher discussion: A model answer must fit the real quantities, units, and allowed inputs. Misconception watch: students may get a correct value without explaining what it means.

Summary

Today you used quadratic models to interpret maximums, minimums, intercepts, and meaningful domains.

Different forms reveal different features: factored form shows zeros, vertex form shows optimum values, and graphs show overall behavior.

A common error is accepting every algebraic solution without checking context.

Strong understanding means you can connect equations, graph features, units, and reasonableness in a final conclusion.

Common Mistakes

  • Using the wrong feature. Why it is tempting: vertices and intercepts both stand out. What to check instead: match the feature to the question.
  • Accepting negative time or length. Why it is tempting: equations may produce negative roots. What to check instead: check domain restrictions.
  • Forgetting units. Why it is tempting: the coordinate pair looks complete. What to check instead: state seconds, feet, dollars, or square units.
  • Maximizing with an intercept. Why it is tempting: intercepts are easy to find. What to check instead: use the vertex for maximum or minimum.
  • Overextending the model. Why it is tempting: the parabola continues forever. What to check instead: restrict the domain to the realistic situation.
