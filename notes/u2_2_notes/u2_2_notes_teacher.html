Graphing Linear Functions
Graph linear functions by using slope, intercepts, and meaningful points.

Mathematical idea: A line can be built from a starting value and a constant rate of change.

Intro

Directions

Read the introduction aloud with a partner or in a small group. As you read, annotate the text by highlighting important ideas, circling key vocabulary, and writing short notes about connections, questions, or predictions in the margins.

A linear graph shows a relationship as a straight line. The $y$-intercept is the point where the line crosses the vertical axis, and it often represents a starting value. In many contexts, that starting value happens before any repeated change is added.

Slope gives a repeatable move on the coordinate plane. From one point, the ratio tells you how far to move vertically and horizontally to land on another point on the same line. This makes graphing more reliable than guessing where the line should go.

A graph can make predictions, but only when the input values still make sense. A savings graph might be useful for the next few weeks, while a negative number of weeks would not fit the situation. Students should think about the labels before extending a line.

Tables generated from a linear rule can support graphing. Each ordered pair from the table becomes a point, and equal changes in the table should line up on a straight line. If one point does not line up, the table or graph may contain an error.

Contextual graphs need axis labels because the same equation can describe many different situations. A slope of $2$ could mean 2 miles per hour, 2 points per game, or 2 dollars per ticket. The labels tell the story behind the line.

Example 1: Identify Slope and Intercept

A line is written as $y=-3x+4$.

  1. Identify the slope.
  2. Identify the $y$-intercept.
  3. Describe whether the line increases or decreases from left to right.

Teacher answer key: Slope is $-3$; intercept is 4; the line decreases. Misconception watch: require students to name the representation evidence they used.

Example 2: Graph from Slope-Intercept Form

A linear function is written as $y=2x-3$.

  1. Name the starting point on the $y$-axis.
  2. Use the slope to list two more points on the line.
  3. Explain how the graph would change if the intercept became $1$ instead of $-3$.

Teacher answer key: Start at $(0,-3)$; points include $(1,-1)$ and $(2,1)$; changing the intercept shifts the line up. Misconception watch: require students to name the representation evidence they used.

Summary

Graphing linear functions means using slope and intercept information to build or interpret a straight-line relationship.

The main representation rule is $y=mx+b$: $b$ gives the starting point and $m$ gives the repeated move.

A common error is graphing the intercept correctly but using the slope in the wrong direction.

Strong understanding means using the graph, equation, and context labels together to make predictions within a meaningful range.

Common Mistakes

  • Switching $m$ and $b$. Why it is tempting: both are numbers in the equation. What to check instead: identify $b$ as the output when $x=0$.
  • Using slope backward. Why it is tempting: rise and run both appear in a ratio. What to check instead: connect numerator to vertical change and denominator to horizontal change.
  • Ignoring slope sign. Why it is tempting: the fraction may look positive. What to check instead: check whether the line rises or falls left to right.
  • Matching by position. Why it is tempting: lists can seem organized. What to check instead: use feature evidence, not order.
  • Extending without context. Why it is tempting: a line continues forever mathematically. What to check instead: ask whether the input still makes sense.
