Algebra 1 - Unit 0B: Functions and Linear Readinessddd

Essential Standard

Students will represent, interpret, and justify relationships on the coordinate plane by connecting points, axes, intercepts, rates of change, and linear representations to mathematical situations.

Section Outline

Section 0B.1: Coordinate Plane and Graph Interpretation

Learning Target: Students will interpret and create graphs on the coordinate plane while connecting points, axes, intercepts, and graphical information to mathematical situations.

I Can Statements

  • I can construct coordinate graphs that connect points, axes, and intercepts to a situation.
  • I can apply coordinate plane information to interpret locations, trends, and key graph features.
  • I can determine whether a graph feature makes sense for the situation it represents.

Key Representations

  • Coordinate plane graphs
  • Ordered pairs
  • Axis labels and scales
  • Intercepts
  • Contextual graph features

Modeling Opportunities

  • Graph points from a classroom or school-based situation.
  • Choose meaningful axis labels and scales for a data display.
  • Connect intercepts to starting values or break-even situations.
  • Interpret graph features in a short written context.
Section 0B.2: Patterns and Input-Output Rules

Learning Target: Students will identify patterns and represent relationships using input-output rules, tables, words, and symbols.

I Can Statements

  • I can formulate input-output rules from tables, words, and symbols.
  • I can use a rule to predict outputs and extend a pattern.
  • I can build tables that organize inputs, outputs, and repeated change.

Key Representations

  • Input-output tables
  • Verbal rules
  • Symbolic rules
  • Pattern diagrams
  • Ordered pairs

Modeling Opportunities

  • Create a rule for a growing visual pattern.
  • Generate a table from a written situation.
  • Translate a rule between words and symbols.
  • Predict later values using an input-output relationship.
Section 0B.3: Linear Patterns and Rate of Change

Learning Target: Students will recognize constant rate of change in tables, graphs, visual patterns, and situations.

I Can Statements

  • I can design tables and graphs that display constant rate of change.
  • I can extend linear patterns using repeated equal change.
  • I can assess whether a table, graph, or situation has a constant rate of change.

Key Representations

  • Tables with equal input steps
  • Linear graphs
  • Visual patterns
  • Rates of change
  • Contextual change statements

Modeling Opportunities

  • Model steady growth or decrease from a real-world situation.
  • Create a table that makes constant change visible.
  • Use a graph to connect steepness with rate.
  • Check whether a visual pattern grows at a constant rate.
Section 0B.4: Writing Linear Relationships

Learning Target: Students will represent linear relationships using tables, graphs, equations, and contextual descriptions.

I Can Statements

  • I can construct linear relationships from tables, graphs, equations, and contexts.
  • I can apply slope and starting value information to write a linear equation.
  • I can use multiple representations to predict values in a linear situation.

Key Representations

  • Linear equations
  • Tables
  • Graphs
  • Contextual descriptions
  • Slope and starting value

Modeling Opportunities

  • Write equations from everyday constant-rate situations.
  • Build a table and graph from a verbal description.
  • Connect slope and intercepts to contextual meanings.
  • Predict future values with a linear model.
Section 0B.5: Comparing Representations

Learning Target: Students will compare relationships using equations, graphs, tables, visual patterns, and contexts, then justify conclusions.

I Can Statements

  • I can build organized comparisons across equations, graphs, tables, visual patterns, and contexts.
  • I can predict which relationship has greater values or faster change using representation evidence.
  • I can critique conclusions about relationships using features from more than one representation.

Key Representations

  • Equations
  • Graphs
  • Tables
  • Visual patterns
  • Context comparisons

Modeling Opportunities

  • Compare two plans, patterns, or situations using multiple representations.
  • Choose evidence from a table and graph to support a conclusion.
  • Revise a comparison when a representation conflicts with another.
  • Connect faster growth, starting value, and later values in context.
Unit Modeling Themes
  • Coordinate graphs communicate information about mathematical situations.
  • Tables, rules, and symbols organize input-output relationships.
  • Constant rate of change connects visual, numerical, graphical, and contextual patterns.
  • Linear relationships can be represented flexibly across equations, graphs, tables, and words.
  • Meaningful comparisons depend on selecting evidence from multiple representations.

Assessment Methods by DOK Level

DOK 1 - Rungs 1-2

Format: 3-5 minute warm-up, vocabulary check, representation recognition, or exit ticket

Purpose: DOK 1 reveals whether students can read coordinate plane features, use ordered pairs, and access basic representations before more complex linear reasoning.

Assessment Ideas

  1. Coordinate Pairs and Axes Can Be Assessed After Section 0B.1 - Apply coordinate signs and positions; distinguish x- and y-coordinates; connect axis locations to input-output order.
  2. Intercepts and Starting Values Can Be Assessed After Section 0B.1 - Determine x- and y-intercepts from graphs and tables; read starting amounts from context graphs; connect vertical intercepts to situation meanings.
  3. Input-Output Rules Can Be Assessed After Section 0B.2 - Use input-output patterns and rules; complete missing outputs in tables; select tables that match symbolic rules.
  4. Rates and Linear Features Can Be Assessed After Section 0B.4 - Interpret repeated change and linear features; identify slope, fixed amount, and starting value; compare output changes across tables.
What this tells the teacher:
DOK 1 results show whether students can read coordinate pairs, axes, intercepts, input-output rules, and repeated change before producing full linear models. The data separates basic coordinate-language readiness from early rule, rate, and starting-value readiness.
DOK 2 - Rungs 3-4

Format: Mini-quiz, graph/table task, matching task, partner check, or short written task

Purpose: DOK 2 reveals whether students can move between representations and use patterns, rates, and starting values to represent linear relationships.

Assessment Ideas

  1. Graph Tables and Intercepts Can Be Assessed After Section 0B.1 - Construct coordinate graphs from tables and contexts; choose readable axis scales; state intercepts and trends from graphed relationships.
  2. Write and Check Rules Can Be Assessed After Section 0B.2 - Formulate rules from input-output tables; generate tables from symbolic rules; check ordered pairs by substitution.
  3. Constant Rate Procedures Can Be Assessed After Section 0B.3 - Apply constant-rate procedures; test output differences for equal input intervals; predict values from starting amounts and graph points.
  4. Linear Models and Comparisons Can Be Assessed After Section 0B.5 - Solve linear-model comparisons; write equations from tables; compare starting values and rates to find when one plan becomes greater.
What this tells the teacher:
DOK 2 results show whether students can carry procedures across graphs, tables, equations, and contexts without losing the meaning of scale, intercept, rate, or starting value. The data helps pinpoint whether errors come from graph setup, rule formation, constant-rate calculation, or linear comparison.
DOK 3 - Rungs 5-6

Format: Constructed response, model critique, strategy comparison, representation analysis, or discourse prompt

Purpose: DOK 3 reveals whether students can reason across representations, check claims, and use evidence about rate, starting value, and graphical features.

Assessment Ideas

  1. Critique Graph Interpretations Can Be Assessed After Section 0B.1 - Critique graph-interpretation claims; use both intercepts to correct starting-value errors; plan axis scales for readable contextual graphs.
  2. Justify Rule and Pattern Consistency Can Be Assessed After Section 0B.4 - Verify rule and pattern consistency; locate mismatches in tables; justify linearity and later predictions with repeated differences and rules.
  3. Compare Representations with Evidence Can Be Assessed After Section 0B.5 - Construct representation-consistency checks; compare graph, table, context, and equation evidence; revise mismatches using slope and starting value.
  4. Compare Claims and Methods Can Be Assessed After Section 0B.5 - Assess claims about competing linear plans; compare table and rate-based methods; reason from equation structure when outputs can or cannot match.
What this tells the teacher:
DOK 3 results show whether students can use evidence to critique claims, verify representation consistency, and compare methods rather than relying on surface features. The data reveals whether students can justify intercept meanings, rule-table mismatches, slope/start evidence, and competing-plan conclusions.
DOK 4 - Rungs 7-8

Format: Brief performance task, modeling task, critique-and-revise task, design task, or capstone synthesis task

Purpose: DOK 4 reveals whether students can synthesize coordinate plane reasoning, input-output patterns, and linear representations in a short modeling or revision task.

Assessment Ideas

  1. Create Coordinate Models Can Be Assessed After Section 0B.4 - Design coordinate-plane models under constraints; coordinate contexts, ordered pairs, graphs, and intercepts; justify agreement across representations.
  2. Revise and Synthesize Rules Can Be Assessed After Section 0B.4 - Create corrected and complete rule representations; revise one-error tables into constant-rate patterns; synthesize tables, rules, graphs, and constraints.
  3. Design Linear Models Can Be Assessed After Section 0B.5 - Develop realistic linear models and domains; connect context, equation, table, graph, and intercept; recommend representations that best show constant rate.
  4. Compare and Recommend Plans Can Be Assessed After Section 0B.5 - Evaluate competing reward plans; revise one-week comparison claims; recommend short-term and long-term choices using starting value, rate, and time evidence.
What this tells the teacher:
DOK 4 results show whether students can design, revise, and defend constrained linear models across context, table, graph, and equation forms. The data indicates readiness for independent modeling, representation choice, and short-term versus long-term comparison decisions.

Summative Assessment Plan

Unit 0B Summative Assessment Early-Course Readiness Checkpoint

Assessment is designed for a 45-minute paper-only time limit and is built around balanced evidence rather than a fixed question-count breakdown. Each version includes a mix of coordinate graph interpretation, input-output tables, pattern rules, rate of change tasks, linear representation tasks, and comparison prompts aligned to the unit learning targets and Depth of Knowledge expectations.

Assessments emphasize interpreting coordinate plane features, creating tables and graphs, recognizing constant rate of change, writing simple linear relationships, and using representation evidence to make conclusions.

Summative assessments are intentionally focused on DoK 1-3 evidence within a paper-only environment. Extended DoK 4 modeling, critique, design, and performance tasks are assessed separately through formative assessments, collaborative activities, investigations, discourse tasks, and performance assessments rather than embedded directly into the summative assessment.

Because of the spiraling course structure, Unit 0B content is revisited through subsequent units via retrieval practice, warm-ups, and formative assessments.

Holistic Rubric

A - Highly Proficient

The student demonstrates strong flexible understanding of coordinate graph interpretation, input-output patterns, rate of change, linear relationships, and representation-based comparisons. Work is accurate, well-organized, and supported by clear reasoning. The student consistently connects equations, graphs, tables, and contexts while justifying conclusions using precise mathematical language.

B - Proficient

The student demonstrates solid understanding of coordinate plane features, pattern rules, rate of change, and linear representations. Most work is accurate, and the student can complete standard and moderately complex tasks independently. Explanations are generally clear, though they may lack some precision or depth.

C - Developing Proficiency

The student demonstrates partial but passable understanding of graph interpretation, input-output relationships, and basic linear readiness concepts. The student can complete many routine tasks but may need support with notation, reasoning, restrictions, representations, or multi-step problem solving. Work may contain errors or incomplete explanations, but there is enough evidence that the student understands the main ideas.

Not Yet - Insufficient Evidence

The student does not yet demonstrate sufficient understanding of coordinate plane interpretation, input-output patterns, rate of change, or linear representations. Work shows major errors, missing reasoning, weak conceptual understanding, or inability to connect representations. The student may need reteaching, additional practice, or reassessment to demonstrate proficiency.
